# CRUD_API

A simple CRUD (Create, Read, Update, Delete) API built with Node.js and TypeScript. This project demonstrates how to build a RESTful API with basic CRUD operations for managing users.

## Features

- Create a new user
- Retrieve all users
- Retrieve a user by ID
- Update a user by ID
- Delete a user by ID

## Installation
Clone the repository and install dependencies:

```sh
git clone https://github.com/arastepa/CRUD_API.git
cd CRUD_API
npm install
```

## Environment Setup
Create a `.env` file in the root directory and configure necessary environment variables according .env.example.

## Running the Application

### Development Mode
```sh
npm run start:dev
```

### Production Mode
To build and run the production bundle:

```sh
npm run start:prod
```
## Testing
Run tests using Jest:

```sh
npm test
```
